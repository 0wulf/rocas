¿Qué es éste lugar?

Éste sitio web estático es una tarea para el curso de Tecnologías y Aplicaciones Web (IIC2513-2023-1) de la Pontificia Universidad Católica de Chile.

¿Qué se muestra aquí?

La página tiene como objetivo presentar una colección de rocas de manera artística y simple, mostrando parte de lo que ha sido la travesía de esta colección.

Contenido

El sitio web está compuesto por tres páginas: Inicio, Acerca de y Rocas. La página de inicio se utiliza de landing page. La página Acerca de presenta información sobre el sitio web y la página de Rocas presenta en detalle la colección de rocas, incluyendo texto e imágenes.

Cada página tiene asociada una hoja de estilos, la cual se encarga de darle un estilo único a cada una. Además existe una hoja de estilos universal, la cual se encarga de darle un estilo a los elementos que son comunes a todas las páginas.
